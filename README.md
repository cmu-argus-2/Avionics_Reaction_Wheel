Equipment:
- DC Power Supply (set to 12V, 0.9A) 
- Nucleo-G474RE mircocontroller board
- MCF8315C motor driver (on MCF8315EVM board, use VBAT not VM)
- 2804 Three Phase Brushless Motor / AS5600 FOC Encoder (DFRobot, SKU: FIT1034)

Arduino IDE Setup:
- Need STM32 MCU based boards by STMircoelectronics board manager installed
- Select Tools > Board part number: > Nucleo G474RE
- Select Nucleo-64 from board & port menu
